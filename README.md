# Modelos-y-Sistemas # Piernitas

Sistema web de gestión para administrar usuarios, clientes, vehículos y turnos.

---

## Descripción

Piernitas es una aplicación web desarrollada con PHP y MySQL que permite organizar información de manera eficiente mediante una base de datos.

Este proyecto fue realizado con fines educativos.

---

##  Funcionalidades

* Registro e inicio de sesión
* Gestión de clientes
* Administración de vehículos
* Sistema de turnos

---

##  Tecnologías

* HTML
* CSS
* JavaScript
* PHP
* MySQL

---

## Instalación

1. Descargar o clonar el repositorio
2. Colocar en `htdocs` (XAMPP)
3. Crear base de datos `piernitas`
4. Importar `base_de_datos.sql`
5. Ejecutar en:
   http://localhost/piernitas
